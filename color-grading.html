Color Grading got easy

The Color Grading Page in DaVinci Resolve is a powerful tool to adjust and enhance the look of your footage. Hereâ€™s a detailed description of its layout, tools, and shortcuts.

1. Overview of the Color Grading Page

Top Toolbar

  • Timeline: Located at the top of the screen, it shows the clips in your project for easy navigation.
  • Gallery: Found on the left-hand side of the top toolbar. Use it to save and apply stills for color reference.

Nodes Panel (Center-Top)

Displays the node tree, allowing you to layer color corrections and effects.

Shortcut: Shift+S (Add Serial Node)

    Alt+O (Add Parallel Node)

Viewer (Center)

Shows the current clip being graded with the applied adjustments.

Right-click for options like Display LUT or Split-Screen.

Scopes (Bottom-Right)

Located at the bottom-right corner, scopes like Waveform, Parade, Vectorscope, and Histogram provide visual data about your image.

Shortcut: Ctrl+Shift+W (Toggle Waveform)

Toolbar (Below Viewer)

  • Primary Wheels: Includes Lift, Gamma, Gain, and Offset wheels for basic color correction.
    Shortcut: Shift+Home (Reset Wheels)
  • Curves: Includes Custom Curves, Hue vs Hue, Hue vs Saturation, etc.
    Shortcut: Alt+C (Open Curves)

Keyframes Panel

Found at the bottom left, used to animate color changes over time.

Power Windows

Create custom masks for targeted adjustments. Located in the toolbar under the "Window" icon.

Shortcut: Alt+W (Activate Power Window)

3. Essential Tools for Beginners

  • Lift, Gamma, Gain, Offset: Located under the Color Wheels, these adjust brightness and contrast for shadows, midtones, highlights, and overall levels.
  • Curves: Found next to Color Wheels, they provide precision for tone and color adjustments.
  • Scopes: Located at the bottom-right, they help monitor changes visually. For example, the Parade shows RGB balance, and the Vectorscope ensures skin tones are accurate.

Workflow Example

1. Select a Clip

Click on the clip in the timeline.

2. Apply Primary Correction

Use Color Wheels for initial brightness and contrast adjustments. Use Shift+Home to reset changes if needed.

3. Add Nodes

Right-click in the Nodes Panel and add a new Serial Node (Shift+S) for secondary color correction.

4. Refine with Curves

Open the Curves panel (Alt+C) for detailed control.

5. Use Scopes

Monitor adjustments using the Waveform and Vectorscope (Ctrl+Shift+W).

6. Mask Areas

Use Power Windows (Alt+W) for targeted corrections.

7. Apply LUTs

Right-click in the Viewer and select LUTs to stylize your footage.
